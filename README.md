# streamlit-study

Learn python by streamlit。机器人聊天开发用到了streamlit和langchain。

## 1. streamlit 安装

```bash
pip install streamlit
```

## 2. streamlit 使用

```bash
streamlit run app.py
```

## 3. langchain 使用

```bash
pip install langchain
```

## 4. 项目依赖安装

```bash
pip install -r requirements.txt
```

## 5. chat_robot运行

需要设置环境变量DASHSCOPE_API_KEY
